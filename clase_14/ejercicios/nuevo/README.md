# Simulador REST - Mini proyecto
Proyecto pequeño para practicar URIs, métodos HTTP y respuestas JSON sin necesidad de backend real. Abre `index.html` en tu navegador para probarlo.

## Archivos
- `index.html` — Interfaz gráfica
- `styles.css` — Estilos (pastel, minimal)
- `app.js` — Lógica que simula la API y muestra resultados
- `README.md` — Este archivo

## Uso
1. Descomprimí el ZIP.
2. Abrí `index.html` con el navegador (doble click).
3. Interactuá con los botones para ver respuestas simuladas y JSON.

> Si querés, puedo convertirlo en una versión que use un backend real con JSON Server o desplegarlo en Vercel/GitHub Pages.
